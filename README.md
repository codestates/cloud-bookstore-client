## Before Start
1. npm install 을 통해서 의존성 모듈을 설치 합니다.
    * npm install --legacy-peer-deps (react-scroll과 use-state-with-callback의 버젼이 현재 react 버전이 맞지 않아 다음과 같은 방법으로 설치 부탁드립니다)
2. npm start 명령어를 실행하면, localhost 3000포트로 연결됩니다.
